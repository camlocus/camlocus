# CamLocus - White-label Cloud Video Surveillance Platform

**CamLocus** is a **white-label cloud video surveillance platform** designed for secure remote access, recording, and management of IP cameras. It enables businesses to stream and store footage off-site without relying on vendor-specific cloud services. CamLocus supports RTSP-compatible devices and integrates with popular camera brands such as **Hikvision**, **Dahua**, **Axis**, **Reolink**, and **Uniview**.

The platform allows centralized monitoring through a browser interface, eliminating the need for complex infrastructure setup. It is ideal for **installers**, **integrators**, and **companies** managing multiple locations. CamLocus helps extend existing **CCTV systems** into a flexible and scalable cloud environment.

## Features

- **White-label CCTV Software**: Fully customizable and branded to meet your specific needs.
- **Multi-language Support**: Available in multiple languages for global accessibility.
- **Use Any Camera**: Supports RTSP-compatible devices, enabling you to use any camera brand or model.
- **Centralized Viewing**: Access all your camera feeds and manage surveillance from a single, easy-to-use browser interface.
- **Source Code Access**: Get access to the full source code for complete customization.
- **CamLocus API**: Integrate with other systems and expand your surveillance solution using the CamLocus API.
- **Users Administration**: Manage and control user access with unlimited users and customized roles.
- **Advanced Playback**: Easily review footage with advanced playback controls, including speed and frame-by-frame features.
- **Analytics**: Leverage video analytics for motion detection, people counting, and more.
- **4G/5G Support**: Seamlessly stream and store footage using mobile networks (4G/5G) for remote installations and monitoring.
